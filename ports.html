Lyamde Squad Ports

Lyamde Squad Ports is a project focused on bringing Source engine games to various platforms, including Android, Windows, and Linux. The project boasts cross-platform compatibility and a functional multiplayer experience, all while maintaining closed-source code to avoid issues with Valve.
